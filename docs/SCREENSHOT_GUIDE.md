# Screenshot Guide

Use these shots for the README, Devpost, or final hackathon submission.

## Required Shots

- Landing page hero with the terminal preview visible
- Debugging arena after a challenge starts
- Correct-answer state with XP and feedback visible
- Dashboard with progress stats
- Leaderboard after submitting a score
- AI Helper showing fallback or Gemini-ready state

## Suggested Capture Settings

- Desktop: 1440 x 900
- Mobile: 390 x 844
- Browser zoom: 100%
- Theme: default dark Arena theme
